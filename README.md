This folder contains code for deploying a machine learning prediction model built for Employee Attrition Prediction using IBM HR Data.
The folder contains the app.py file and the encoder.pkl, scaler.pkl, and model.pkl files.
Additionally, it contains a requirements.txt file to help the app install the necessary libraries to run.
This is a group effort with the name "Team Sentinel" under the Data Science Track, Cohort 3 of TechCrush 3-month Bootcamp 2025.
